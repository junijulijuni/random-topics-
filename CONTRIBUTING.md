This Wiki / Knowledge Base / Info Dump is organized with <u>folders</u> and <u>markdown</u> files. Topics are organized by category and subcategories (folder structure) and Topic pages 
collect the information for a specific topic.

GitHub works by using Markdown files. Markdown is a specific formatting syntax for text files (similar to rtf, LaTeX, etc). In order for a Markdown page to be readable and look nice it
requires the author to have a basic understanding of how to format using Markdown.
